# Cancer Care Platform

A production-ready MERN stack web application for cancer patients and their care teams, optimized for performance, accessibility, and security.

### MVP Features
- **Authentication & Accounts**: Secure sign-up/sign-in with role-based access

- **Medication & Symptom Tracker**: Log medications and symptoms with charts
- **Appointment Management**: Calendar view with reminders
- **Secure Document Vault**: Encrypted file storage with sharing
- **Care Team Directory**: Clinician directory and community forum
- **Audit Logging**: Comprehensive activity tracking

- FAQ answer bot
- Smart appointment prep
- Treatment plan simplifier
- Mood and wellness assistant
